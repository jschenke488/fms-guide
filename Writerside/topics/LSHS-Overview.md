## Terminology

| Term                | Definition                                                                          |
|---------------------|-------------------------------------------------------------------------------------|
| Driver Station (DS) | Computer running software to communicate between robots and drivers.                |
| WPA Keys            | Wi-Fi passwords automatically generated for robots to connect to the field radio.   |
| WPA Kiosk           | Computer used to program a robot radio for use with the field.                      |
| Field Radio         | Robots connect to Wi-Fi running on this device to communicate with driver stations. |
